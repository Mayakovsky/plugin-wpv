# 2. Custom Logic Check: The Content Flywheel

## 2.1 Why Custom Logic Must Exist

The Daily 10 cron builds our inventory. But the ecosystem produces 50–100 whitepaper-backed projects daily. We only catch 10. Custom Logic Check turns customers into a discovery engine: we get paid $1.60 net to verify, the whitepaper enters our database permanently, every future query is cached at near-zero cost.
